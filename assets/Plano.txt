# Plano Frontend - CovidDataFlow

## 🎯 Objetivo
Criar uma interface profissional em React + TypeScript que:
- Mostre inicialmente um artigo explicativo sobre a COVID.
- Tenha um botão chamativo que, ao ser clicado, deslize a tela para a lateral.
- Exiba uma nova seção com cor diferente, contendo:
  - Um resumo do artigo.
  - Um gráfico dinâmico alimentado pelo banco de dados do backend Python.

## 📊 Fluxo de Funcionamento
1. **Tela inicial (Home.tsx)**  
   - Renderiza `Article.tsx` com texto explicativo.  
   - Mostra `Button.tsx` chamativo (“Ver Resumo”).  

2. **Interação**  
   - Ao clicar no botão, `SlidePanel.tsx` aplica transição lateral.  
   - Surge `Summary.tsx` com cor diferente e texto resumido.  
   - `Summary.tsx` consome dados da API (`services/api.ts`) e renderiza gráfico.  

3. **Gráfico**  
   - Alimentado pelo banco de dados via backend Python.  
   - Mostra casos, óbitos ou outros indicadores relevantes.
